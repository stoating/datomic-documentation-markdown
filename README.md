# Datomic Documentation (Markdown)

This repository contains the Datomic documentation as available on [docs.datomic.com](https://docs.datomic.com/datomic-overview.html) as of April 19, 2026, reformatted into Markdown.

The goal is to make the documentation available in a structured, file-based format for easier ingestion by programs and tools (e.g. search indexers, AI assistants, static site generators).
