## Code Samples

- [Datomic Tutorials](https://github.com/Datomic/datomic-tutorials): code repository for the tutorials blog
- [Day of Datomic](https://github.com/cognitect-labs/day-of-datomic-cloud): code for the [Day of Datomic series](https://www.youtube.com/watch?v=yWdfhQ4_Yfw)
- [Mbrainz importer](https://github.com/Datomic/mbrainz-importer): import the Mbrainz data subset into your Datomic Cloud System
- [Ion event example](https://github.com/Datomic/ion-event-example): heavily commented sample [Datomic Ions](../../07-datomic-cloud-ions/01-ions-overview/ions-overview.md) application
- [pedestal.ions](https://github.com/pedestal/pedestal.ions): a Pedestal interceptor chain provider for Datomic ions

### Third-Party Examples

- [Cognito Authentication for Datomic Cloud](http://www.jarrodctaylor.com/posts/Cognito-Authentication-For-Datomic-Cloud/)
- [Schema cartographer](https://github.com/JarrodCTaylor/schema-cartographer): visualize, navigate, create, edit, and share the relationships that exist in your Datomic schema
- [Conformity](https://github.com/avescodes/conformity): a Clojure/Datomic library for idempotently transacting norms into your database – be the schema, data, or otherwise
- [MySQL to Datomic](https://github.com/thosmos/mysql-to-datomic): MySQL to Datomic
- [Datomic basics](https://github.com/benkamphaus/datomic-basics): Presentation and walkthrough for Datomic Basics talk at the Clojurians meetup
- [Spectomic](https://github.com/Provisdom/spectomic): generate Datomic or Datascript schema from your Clojure(Script) specs
- [Datomic-Service](https://github.com/jdhollis/datomic-service): a [Terraform](https://www.terraform.io) wrapper for Datomic Cloud
- [Datomic Cloud Primer](https://github.com/clj-nakano/datomic-cloud-primer): japanese introduction to Datomic Cloud

### Ions

- [Heroi.cc](https://github.com/niamu/heroi.cc): Clojure (Datomic ion) source code for the [web app](https://heroi.cc)
- [Ion and AWS AppSync example](https://github.com/hlprmnky/ion-appsync-example): wiring up [Datomic Cloud ions](../../07-datomic-cloud-ions/01-ions-overview/ions-overview.md) to [AWS AppSync+](https://aws.amazon.com/appsync/)
- [Ions shopping list](https://github.com/viesti/ions-shopping-list): playground for Datomic ions
- [Valence](https://github.com/eoliphan/valence): a small library of useful utilities for Datomic ions-based apps
- [Ion-web](https://github.com/MageMasher/ion-web): small PoC of invoking a Datomic oon directly from the browser, no API Gateway required
